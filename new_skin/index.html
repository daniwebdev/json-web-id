JSON.WEB.ID

JSON Example For Multi Perpose Application

Option

How to use

Header: x-api-key (optional)

if you fill in the key, then your data will not be mixed with other users
  • [GET] https://json.web.id/app/{namespace?}?uuid={uuid?}

    Params

    Name Type Description
    namespace string your own namespace if you want to make some apps or modules
    uuid string if you want get with spesific uuid

    Response Example
